📚 Hub de Leitura - Swagger Helper

Facilite seus testes na API com autenticação automática!

🚀 Como usar o Swagger com autenticação:

1 Escolha uma credencial abaixo e clique em "Login Rápido"
2 Copie o campo "token_for_swagger" (apenas o hash, sem "Bearer")
3 Vá para o Swagger e clique no botão "🔒 Authorize"
4 Cole APENAS o hash no campo "Value" - Swagger adiciona "Bearer" automaticamente
5 Clique "Authorize" e depois "Close"
⚠️ IMPORTANTE para estudantes:
• No Swagger: Use apenas o hash (ex: "eyJhbGciOiJIUzI1NiIs...")
• No Postman: Use o token completo (ex: "Bearer eyJhbGciOiJIUzI1NiIs...")
• O Swagger adiciona "Bearer " automaticamente!
